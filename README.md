# NodeJS Middleware Extract (Example)

Minimal Node.js + Express example app with routes, EJS views, partials and styles.

Quick start

```bash
npm install
# development with automatic reload
npm run dev
# or start production server
npm start
# open http://localhost:3000
```

Files of interest:

- [app.js](app.js)
- [routes/index.js](routes/index.js)
- [views/index.ejs](views/index.ejs)
- [views/about.ejs](views/about.ejs)
- [views/partials/head.ejs](views/partials/head.ejs)
- [public/css/style.css](public/css/style.css)
 - [views/layout.ejs](views/layout.ejs)
 - [views/contact.ejs](views/contact.ejs)
